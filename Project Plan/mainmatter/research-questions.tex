\chapter{Research Questions and Project Objectives}
\label{chapter:research-questions}

\section{Main Research Question}

\begin{quote}
\textit{How can we design and build an AI-powered monitoring system that works together with Cilium-based network security to enable proactive, automated management of Kubernetes infrastructure?}
\end{quote}

\section{Sub-Questions}

To answer this, we'll break it down into five focused questions, each looking at a different piece of the puzzle:

\begin{enumerate}

    \item \textbf{What are the main problems with traditional infrastructure monitoring and network security in Kubernetes?}

    \item \textbf{Which AI and machine learning techniques work best for detecting anomalies and predicting problems in infrastructure and network metrics?}

    \item \textbf{How does Cilium's eBPF-based approach improve network security and observability compared to traditional Kubernetes networking?}

    \item \textbf{What kind of integration architecture do we need so the AI monitoring system can use Cilium and Hubble data to provide correlated security and performance insights?}

    \item \textbf{How can we test and validate that the integrated solution actually works well, doesn't slow things down too much, and improves security?}

\end{enumerate}
